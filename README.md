# Wim Hof Breathing App

Web-приложение для дыхательной практики по методу Вима Хофа.
Запускайте в Mini App через Telegram бота @breathingapp_bot

## Особенности

- Интерактивная анимация дыхания с визуальным таймером
- Настраиваемое количество раундов (1-10)
- Регулируемое время задержки дыхания
- Настройка длительности вдоха/выдоха
- Звуковое сопровождение практики
- Светлая и тёмная тема
- Поддержка русского и английского языков
- Интеграция с Telegram Mini Apps
- Прогрессивное увеличение времени задержки между раундами

## Как использовать

1. Нажмите **Начать** для старта практики
2. Следуйте инструкциям на экране: вдох → выдох (30 циклов)
3. После 30 вдохов — задержка дыхания
4. Глубокий вдох и 15-секундная задержка
5. Глубокий выдох — раунд завершён

## Настройки

- **Количество раундов** — от 1 до 10
- **Время задержки** — базовое время в секундах (увеличивается каждый раунд)
- **Длительность вдоха** — от 1 до 2.5 секунд

## Технологии

- Vanilla JavaScript
- CSS3 анимации
- Telegram Web App SDK
- Wake Lock API (экран не гаснет во время практики)

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 3000
```
